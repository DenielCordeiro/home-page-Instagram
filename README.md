# Instagram Home Page Clone 📸

![Status](https://img.shields.io/badge/Status-Concluído-success)
![GitHub language count](https://img.shields.io/github/languages/count/DenielCordeiro/home-page-Instagram)

Este é um projeto de front-end que replica a interface da página de login do Instagram. O objetivo principal foi praticar a estruturação de layouts semânticos com HTML5 e a utilização de **SASS** para uma estilização mais profissional e organizada.

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes ferramentas:

* **HTML5**: Estruturação de conteúdo.
* **SASS (Syntactically Awesome Style Sheets)**: Para uma estilização avançada, utilizando variáveis, aninhamento e organização de arquivos.
* **Flexbox**: Utilizado para garantir o alinhamento correto dos componentes da página.

## 🛠️ Como executar o projeto

Você pode visualizar o projeto localmente seguindo um dos métodos abaixo:

### Opção 1: Download Direto (Mais simples)
1. No topo desta página, clique no botão verde **Code** e selecione **Download ZIP**.
2. Extraia o arquivo `.zip` em uma pasta no seu computador.
3. Entre na pasta extraída e dê um clique duplo no arquivo `index.html`. O projeto abrirá automaticamente no seu navegador.

### Opção 2: Via VS Code (Recomendado para Devs)
1. Clone o repositório ou baixe os arquivos.
2. Abra a pasta do projeto no **VS Code**.
3. Com a extensão **Live Server** instalada, pressione `Alt + L` seguido de `Alt + O` (ou clique em "Go Live" na barra inferior).
